# Comunidade DevJr

Este projeto visa colaborar com a comunidade #ComunidadeDevJr, promovendo o desenvolvimento pessoal e profissional em programação.

## Foco

O principal objetivo é eliminar o "vício" de depender excessivamente da IA para escrever código, incentivando o aprendizado ativo. Inspirado pela #ComunidadeDevJr, estabeleci como meta retomar os estudos em linguagens que já domino e explorar novas áreas.

Este projeto consiste em um blog onde todos podem publicar seus projetos, permitindo-me desenvolver, praticar e aprofundar meus conhecimentos em PHP.

## Tecnologias Atuais

Atualmente, desenvolvo em:

- PHP
- SQL ( Básico essencial de um CRUD )
- C# (Unity Engine para 2D, 3D e AR)
- Python (Análise de dados)

## Planos Futuros

Pretendo expandir meus horizontes para outras tecnologias, incluindo:

- Criação de bots para Discord utilizando Python.
- Ferramentas e conteúdo sobre análise de dados com Python (com e sem Notebook).
- Desenvolvimento de apps e games na Unity e em projetos AR.
- alguns pequenos projetos web/discord com foco em FiveM


## Sobre o Projeto - Protótipo
### Blog para publicações sobre desenvolvimento de projetos
---
- `Criar conta e logar ( De forma simples, sem Auth (*pelo menos no início*) )`
- `Se trata de um blog onde pode ser possível postar E ver outras postagens de Dev's`
- `Poderá ser possível criar nova postagem com título, descrição detalhada e até adicionar alguma imagem ou vídeo ( de forma básica via Link )`
